Welcome to strongMan Client Edition!

strongMan is a management interface for the strongSwan VPN solution. StrongMan provides a user friendly graphical interface to configure and establish IPsec connections.

Notice the button on top right? Change to server-mode and back.

Getting started

  • Create a new connection with your credentials and establish a IPsec tunnel in the connections section.
  • Upload certificates to establish an IPsec tunnel in the certificates section.

Welcome to strongMan Server Edition!

strongMan is a management interface for the strongSwan VPN solution. StrongMan provides a user friendly graphical interface to configure and establish IPsec connections.

Notice the button on top right? Change to client-mode and back.

Getting started

  • Upload certificates to establish an IPsec tunnel in the certificates section.
  • Create a new site-to-site or remote access connection and start or load an IPsec tunnel in the server-connections section.
  • Create and manage EAP Secrets to let users authenticate at your server in the EAP secrets section.
  • Create and manage Pools to use them in the connections setup in the pools section.
